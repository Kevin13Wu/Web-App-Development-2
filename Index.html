17. Add two new messages to the SignIn view (Assignment)

For this assignment you will be writing (and copy/pasting) all of the code yourself. Read this assignment carefully.

You will be adding two new messages to the SignIn view. You'll want to use code from existing views and controllers as a start. You'll need to modify the code you copy to match the requirements below.

  1. Both fields on the SignIn view should be required. If the user doesn't enter information in both of these fields, add a message that all fields are required.
  2. If the login fails (with the UserID and Password entered) add a message “Invalid UserID and/or Password. Please try again."

For both of these you will need to be sure to set the ActionType property to the appropriate value in the SignIn “post” ActionResult method in the ProfileController.cs. You’ll also need to add two “if” statements in the SignIn.cshtml.

This will be very similar to the message in the SignUp view and SignUp "post" ActionResult method.
